# Content Strategy

Plan content that drives traffic, builds authority, and generates leads.

## Two Content Types

**Searchable** — captures existing demand
- Use-case content ("best tool for X")
- Hub-and-spoke (one pillar + 10 cluster posts)
- Template libraries
- Case studies with specific results

**Shareable** — creates demand
- Thought leadership (contrarian takes)
- Original research / data
- Expert roundups
- Behind-the-scenes transparency

**Rule:** Build searchable as foundation. Add shareable to accelerate.

---

## Process

### Step 1 — Gather Context
- What's the product and who is the customer?
- Current traffic / leads from content (if any)?
- Available resources (writer, designer, video)?
- Competitors' content strengths?

### Step 2 — Define Content Pillars (3-5 topics)

Each pillar must satisfy:
1. Core to your product's value
2. Your audience searches for it
3. You can credibly own it
4. Competitors haven't dominated it

### Step 3 — Map to Buyer Journey

| Stage | Intent | Content Type | Example |
|-------|--------|--------------|---------|
| Awareness | "What is X?" | Educational blog, video | "What is cortisol and why does it matter?" |
| Consideration | "Best X for Y" | Comparison, roundup | "Best supplements for stress in 2025" |
| Decision | "X vs Y", "X reviews" | Comparison, case study | "Product A vs Product B" |
| Implementation | "How to X" | Tutorial, checklist | "How to take ashwagandha correctly" |

### Step 4 — Prioritize Ideas

Score each idea:

| Factor | Weight |
|--------|--------|
| Customer impact (solves real problem) | 40% |
| Content-market fit (we can execute well) | 30% |
| Search potential (monthly searches) | 20% |
| Resource requirement (effort to produce) | 10% |

### Step 5 — Output

- 3-5 content pillars with rationale
- 10-20 priority topics with target keywords and buyer stage
- Topic cluster visualization
- 90-day editorial calendar

## Research Sources

| Source | What to find |
|--------|-------------|
| Keyword tools (Ahrefs, SEMrush) | Search volume, difficulty, gaps |
| Sales call recordings | Language customers use |
| Support tickets | Problems customers can't solve alone |
| Reddit, forums | Real questions in real language |
| Competitor content | What ranks, what gaps exist |
| Customer surveys | Jobs to be done |

---

## Content Quality Standards

- Match search intent exactly (informational vs transactional)
- Comprehensive coverage (answer the question fully)
- Customer language (use words from research, not marketing copy)
- Specific over vague (numbers, examples, named tools)
- Updated (set calendar reminders to refresh key posts)

## Self-Check

Before presenting output, verify:
- [ ] 3-5 content pillars defined — each satisfies all 4 criteria (product value, searchable, ownable, gap exists)
- [ ] Buyer journey mapping complete — Awareness, Consideration, Decision, and Implementation stages covered
- [ ] 10-20 prioritized topics with target keywords and buyer stage labeled for each
- [ ] Scoring framework applied — each topic scored on Customer impact, Content-market fit, Search potential, Resource requirement
- [ ] 3 quick-win recommendations identified (highest impact + lowest resource + earliest buyer stage)
- [ ] 90-day editorial calendar included

If any check fails → fix it before presenting.
